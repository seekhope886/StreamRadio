# StreamRadio
\n
2026.04.16 by Halin\n
\n
Stream audio by ESP32 c3 super mini\n
使用ESP32 C3 super mini + Max98357a + SSD1306 I2C 128*64\n
解碼函式庫採用arduino-audio-tools-1.2.2\n
功能:\n
  1.從web server設定電台並操作新增刪除編輯播放停止\n
  2.電台串流格式支援MP3以及AAC兩種\n
設置:\n
  1.更改自己的WiFi聯線帳號密碼後編譯並載入ESP32\n
  2.開啟同區域網路的手機平板電腦瀏覽器開啟ESP32的IP後設置電台即可\n
